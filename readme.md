# 📚 데이터 엔지니어링 디자인 패턴 스터디

데이터 파이프라인의 설계 원칙을 이해하고, 실무에서 발생하는 복잡한 데이터 문제를 해결하기 위한 디자인 패턴을 학습하는 저장소입니다.

- **도서명**: 데이터 엔지니어링 디자인 패턴
- **주요 내용**: 10개 장에 걸친 70가지 데이터 엔지니어링 디자인 패턴 상세 분석

## 🎯 스터디 후 기대 역량
1. **아키텍처 설계 능력**: 데이터 수집~저장 단계별 최적 패턴 선택 및 조합 설계
2. **데이터 무결성 보장**: 멱등성 및 오류 관리 패턴 적용으로 유실 없는 파이프라인 구축
3. **시스템 최적화**: 파티셔닝 및 조회 성능 최적화로 대규모 데이터 처리 효율 극대화
4. **운영 안정성 강화**: 관찰 가능성 패턴 기반 장애 선제적 대응 체계 구축

## 🛠️ 운영 규칙
1. **발제**: 매주 지정된 발제자가 해당 범위 패턴 중 실무 유용 사례 심화 분석 및 공유
2. **개조식 정리**: 모든 요약 문서는 가독성을 위해 불필요한 조사 생략 및 작동 방식 근거 기술
3. **실무 연계**: 학습 패턴을 현재 스택(Spark, Kafka, AWS 등)에서 구현할 방법 및 한계점 토론
